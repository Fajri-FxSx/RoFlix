# Responsive React Movie Website

 URL: https://roflix.vercel.app/
 
 A responsive website with API to watch the lates movies and tv shows.


## Technologies

- ReactJS - Hooks.
- SCSS
- Libraries such as swipperjs, clsx, query-string, axios, ...
